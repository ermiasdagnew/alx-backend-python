# 0x03. Unittests and Integration Tests

## Project Overview

This project focuses on writing **unit tests** and **integration tests** for Python code, specifically for:

- `utils.py` → generic utility functions
- `client.py` → GitHub organization client

The project demonstrates **mocking**, **parameterization**, and **fixtures** for testing code that interacts with external services like HTTP APIs.
